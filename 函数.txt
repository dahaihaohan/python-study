
函数可以有参数，也可以没有，但必须保留括号
	1.
		def <函数名>():
			<函数体>
			return <返回值>
	2.
		def <函数名>(<非可选参数>,<可选参数>)：
			<函数体>
			return <返回值>
		如：
			def fact(n,m=1):
				s=1
				return s//m
	3.
		函数定义时可以设计可变数量参数，既不确定参数总数量
			def <函数名>(<参数>,*b):
				<函数体>
				return <返回值>
函数可以返回0或多个结果：
	如：return a,b,c;
		返回元组类型
		也可以使用a,b,c=fact()来接收，分别赋值
		
在函数中使用全局变量时:
	需要使用global保留字，来声明是全局变量
	
lambda:
	lambda函数返回函数名作为结果：
	-lambda函数是一种匿名函数，即没有名字的函数
	-使用lambda保留字定义，函数名是返回结果
	-lambda函数用于定义简单的、能够在一行内表示的函数
	<函数名> = lambda <参数>:<表达式>
